# WebRTC - Proyecto Camera Stream

Sistema básico de transmisión de video en tiempo real usando WebRTC, Node.js y Socket.io.

---

## ¿Qué hace este proyecto?

Permite transmitir la cámara de un dispositivo (celular o PC) hacia otro dispositivo en tiempo real mediante navegador web.

-  Un dispositivo actúa como cámara
-  Otro dispositivo actúa como visualizador
-  Funciona en red local o con túnel (ngrok)

---

## Tecnologías usadas

- Node.js
- Express
- Socket.io
- WebRTC (RTCPeerConnection)

---

## Instalación

npm install

## Ejecución y uso

- Iniciar sevidor: node server.js
- Camara: http://localhost:3000/camera.html
- Viewer: http://localhost:3000/viewer.html
